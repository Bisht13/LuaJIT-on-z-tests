# LuaJIT-on-z-tests
Here are some basic tests provided for verifying the functioning of LuaJIT on s390x architecture.

There are 2 tests, 

* The first one is for testing basic instructions and instruction format of s390x.
* The second one is for testing the speed of jitted and non-jitted code. This test is the s390x version of the test provided in the [unofficial documentation of DynASM](https://corsix.github.io/dynasm-doc/tutorial.html).

## How to run
Assuming you are on a s390x instance,
* Clone latest LuaJIT from [LuaJIT's Github](https://github.com/LuaJIT/LuaJIT)

```bash
$ git clone https://github.com/LuaJIT/LuaJIT.git
```
* Go to the dynasm directory

```bash
$ cd LuaJIT/dynasm
```
* Clone this repository

```bash
$ git clone https://github.com/Bisht13/LuaJIT-on-z-tests.git
```
* Enter the directory, give run.sh permission to run and run it

```bash
$ cd LuaJIT-on-z-tests
$ chmod +x ./run.sh
$ ./run.sh
```
